Styling details

Typography scale: H1 very large (~44–52px desktop), card titles ~24px, price numbers very large (~56–64px), subtitles and lists ~14–16px.

Colors:

Background: #0B0F0E to #0F1513 gradient glow center top.

Card base: very dark green (#0F1A17) with subtle vertical gradient; border: 1px solid rgba(100, 255, 200, 0.06); shadow: soft, spread to simulate glass depth.

Primary greens: buttons/badges use mid-green (#1DB954 to #27D07D range). Text mostly #EAF8F3; muted text #A5B7AF.

Check icons and accent dots use bright green.

Buttons: 14px/16px text, 52–56px height, full width, rounded-full, gradient fill, hover translate-y-[-1px], active translate-y-[0], focus visible outline.

Badges: small rounded pills, uppercase or semi-bold, green border and faint fill; the discount pills are slimmer.

Dividers: 1px horizontal line, very subtle (rgba white/green at ~0.08).

Accessibility & behavior

All interactive elements have :hover, :focus, :active states.

Cards slightly scale up (1.01) on hover.

Mobile: stack cards with 20–24px gap; reduce price and H1 sizes proportionally; buttons remain full width.
